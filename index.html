Primero
¿Qué son?

Una función condicional es aquella que nos permite realizar una acción siempre y cuando se cumpla con una condición determinada por el usuario. Por ejemplo, cuando se quiere saber cuales fueron las ventas de un empleado, caso en el cual Excel sumará sólo las ventas de Ricardo Rave, por ejemplo. El verdadero potencial de excel está fundamentado en el usa de funciones preestablecidas y el uso de restricciones en estas




Segundo
¿Para qué sirven?

Sirven para establecer condiciones y si se cumplen se podrá realizar la acción, pero si no se cumple no se podrá realizar la función, nos sirve en pocas palabras para saber si la indicación dada si se está cumpliendo sin ninguna falla.




Tercero
¿Cómo se utilizan?

Las condiciones se utilizan para modificar el comportamiento de todas las funciones que utiliza excel, con la función SI se pueden crear condiciones para ver si se cumple algo. La función SI de Excel te permitirá decidir que dato, operación o función utilizar en caso de que se cumpla o no determinada condición.



Cuarto
¿Son útiles?

La funcion condicional en Excel. ... Este método ofrece a los usuarios la posibilidad de visualizar grandes o complejas cantidades de datos. La gran ventaja es que toda esa información la recoge en uno a mas cuadro sacando una conclusión de los datos, por ejemplo el promedio de los datos, la suma, la resta, etc.
